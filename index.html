Projects

Pandemic Georgia

Pandemic Georgia is a browser based game inspired by the board game Pandemic. The goal of the game is for a team of 2-4 players moving about the state treating a spreading disease while searching for knowledge to create a cure. The players have 19 days to accomplish this as the disease spreads with each turn. It was built using Node.js, JavaScript, HTML, CSS, PostgreSQL and Express.js.

brewfinder

Brewfinder was conceived to help people find the closest micro-breweries and brewpubs to the their location. It finds the users location using the geolocation web API and then sorts the six closest breweries from the OpenBreweryDB based on their distance from the user. It was built using JavaScript, HTML, CSS and Bootstrap.

PyCatWhy!?!

PyCatWhy!?! is a clicking game about trying to get a cat into its cat carrier. Only issue is the cat hates you. And oranges. And bananas. And both of those are scattered around the yard. The game features the ill-tempered cat, mobile and stationary obstacles, and 10 levels of increasing difficulty. It was built using Python and the Pygame library.
