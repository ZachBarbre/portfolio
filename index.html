About Me

Greetings and Salutations. I'm Zach, and I'm a web developer  from Atlanta.

I'm a licensed Professional Engineer who became entranced with writing code. Solving problems has always my favorite part of engineering and I find the immediate feedback when a programming problem is solved to be immensely pleasing. Because of this I decided to make the jump into a new career. 

I'm a native of the Atlanta area and graduated from Georgia Tech with a degree in Mechanical Engineering. When I'm not slinging code I can be found with my wife in the Appalachian Mountains, hiking the trails, climbing the rocks and paddling the rivers.  You also might find me sitting around a table with friends playing some too complicated board game.

I currently work as a front-end devloper for Perspecta, Inc contracted to GRASP at the CDC. I help build CDC's COVID Data Tracker.

Projects

Nunnally International is a five page marketing website built for the Mergers and Acquisitions firm, Nunnally International. It is designed for speed and SEO. It was built using Sveltve/Sapper.

Jessie Foster Photography

Jessie Foster Photography is a CSS reskin of the Smugmug website of Jessie Foster. It was built using the Smugmug sitebuilder with custom HTML and CSS blocks.

DrinkCast

DrinkCast Live is a video chat app where you are your friends are able to replicate the experience of sitting at a bar playing games. It allows users to create a bar, join a bar, video chat and then play either 'Would you rather' or 'Never have I ever'. It was built with React, OpenTok React, Node.ja, PostgreSQL and Express.js.

Some Hard F***ing workout

Some Hard F***ing workout is a fitness timer that gives the user a hard workout to do and then keeps track of the time spent, current set and current round. If the workout is too hard the user can randomize a new workout. It was built with React.
